# 🪟 MistWindow（胧雾清窗）

> 一扇凝雾的窗。挥挥手，擦亮它。

MistWindow 是一个运行在浏览器端的创意交互艺术项目。它在屏幕上还原了一扇冬日凝雾的玻璃窗——你不需要触碰屏幕，只需在摄像头前挥动双手，系统就会实时将手势化作指尖温热，轻柔抹去层层迷雾，露出背后的画面。

这是一个将手势识别技术包装成视觉解压疗愈的有趣尝试。

---

## ✨ 核心亮点

### 🖐️ 隔空手势擦除

基于 Google MediaPipe 机器视觉，精准追踪手掌 21 个骨骼关键点。在空气中挥挥手，就能实现天衣无缝的「隔空擦窗」。

### 💨 随心而动的擦雾细节

- **真实物理反馈**：张开五指挥动，雾气顺着手指被平行刮出五道清澈指痕；握拳或并拢手掌，则变成大面积的强力抹除。
- **边缘羽化质感**：擦出的轨迹带有微微晕染、边缘毛糙的物理质感，完美模拟手指划过真实水汽时的水痕扩散效果。
- **动态渐进清澈**：白茫茫的厚雾先褪成半透明薄纱，再彻底化为透明，层次感极强。

### 💧 动态物理水滴

玻璃顶部会随机生成重力水滴，顺着玻璃流淌，沿途留下清澈划痕——物理动态非常治愈、极其解压。

### 🎛️ 随心切换的环境气候

| 模式 | 效果 |
|------|------|
| **凝雾模式** (Recover) | 擦干净的地方，过一会儿会像真实冬日车窗一样，慢慢重新结上白雾 |
| **常清模式** (Dissolve) | 擦掉的部分保持永久清澈 |

### 📁 自定义窗外风景

支持一键热切换背景：使用实时摄像头画面，或上传本地任意视频、壁纸作为窗外风景。

### 🙈 极简「净界」

点击右上角按钮，所有按钮面板顺滑隐藏，整个屏幕只剩一扇朦胧的窗，沉浸感拉满。

## 🛠️ 技术内幕

为了在浏览器里跑出最完美的物理直觉，底层做了这些不着痕迹的调校：

- **五指独立拓扑与动态画笔**：根据手掌张开幅度动态改变擦雾画笔半径——手掌张开，擦除线条细腻；手掌收拢，画笔自动聚合。擦雾不再是僵硬点击，而是像真手抚摸擦拭一样自然。

- **多层 Canvas 离屏缓冲**：通过纯前端 Canvas 2D 建立「底片层 → 迷雾层 → 水滴轨迹层」离屏双缓冲管线，将像素级复合擦除的计算压力降到最低，手机端轻松跑满 60 帧。

- **连续轨迹矢量插值**：针对摄像头帧率不稳或手势晃动过快，内置高密度连续轨迹插值，在两帧手势点之间自动补充缺失的擦除路径——雾气被平滑拉出连续直线，绝不留下断断续续的圆点。

- **全平台播放安全**：针对 iOS Safari 等浏览器的严格隐私和自动播放限制，通过首屏「开场门禁锁」机制优雅绕过，实现全平台点开即玩，100% 唤醒摄像头。

## 🚀 快速开始

纯前端实现，无需搭建任何开发环境。

```bash
git clone https://github.com/Ka9e12/MistWindow.git
cd MistWindow
```

双击打开 `index.html` 即可在浏览器中运行。

> 💡 请在弹出的浏览器提示中允许网页获取摄像头权限，否则「隔空擦窗」功能会失效，只能用鼠标拖拽来擦窗户。

你也可以直接在线体验：**[ka9e12.github.io/MistWindow](https://ka9e12.github.io/MistWindow/)**
